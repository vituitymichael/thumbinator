# 🤖 Thumbinator

> *I'll be back... with your 534px × 300px thumbnail.*

A single-file, in-browser tool that resizes any image to Vituity LMS course thumbnail dimensions. Drag, drop, done.

## Features

- Drag-and-drop, click-to-browse, or paste from clipboard
- Four resize modes: **Crop to Fill**, **Fit Width**, **Fit Height**, **Stretch**
- 9-point crop anchor for precise framing
- Custom canvas background color (with screen eyedropper)
- Optional border/stroke
- Zoom + nudge fine-adjust
- Export as JPG, PNG, or WebP with quality control
- Copy straight to clipboard or download (filename auto-prefixed with `[THUMBINATOR]`)

## Use

Open `index.html` in a modern browser. Sign in with your Vituity credentials. Drop an image. Adjust. Download.

## Tech

100% client-side. No server, no uploads, no tracking. Single HTML file.
